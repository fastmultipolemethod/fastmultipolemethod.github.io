DASHMM

The Dynamic Adaptive System for Hierarchical Multipole Methods is a general library implementing hierarchical multipole methods using the asynchronous multi-tasking HPX-5 runtime system. It provides a unified execution on both shared and distributed memory architectures. The package provides built-in implementations of 3D Laplace, Yukawa, and low-frequency Helmholtz potenials.

RECFMM

RECFMM is a program representation and implementation of a recursive scheme for parallelizing the adaptive FMM on shared memory computers using Cilk runtime. RECFMM provides implementations of 3D Laplace and Yukawa potentials.

Reference: B. Zhang, J. Huang, N. P. Pitsianisn, and X. Sun. RECFMM: Recursive Parallelization of the Adaptive Fast Multipole Method for Coulomb and Screened Coulomb Interactions. Commun. Comput. Phys. 20 (2016) 534-550

  • KIFMM
    This kernel-independent FMM was developed by Ling, Biros and Zorin. It replaces the analytic expansions and translations in the FMM with equivalent density representations. The packages applies to second-order constant coefficient non-oscillatory elliptic PDEs.
    Reference: L. Ying, G. Biros, and D. Zorin. A kernel-independent adaptive fast multipole algorithm in two and three dimensions. J. Comput. Phys. 196 (2004) 591-626
  • 2D-WFMM
    2D-WFMM is a wideband FMM package for the 2D Helmholtz equation. The package is parallelized using OpenMP on shared memory computers.
    Reference: M. Cho and W. Cai. A Wideband Fast Multipole Method for the two-dimensional complex Hemholtz equation. Comput. Phys. Commun. 181 (2010) 2086-2090
  • ExaFMM
    ExaFMM is an open-source parallel, GPU-enabled code in the Barba's group.
  • Adaptive Tree Codes
    Collection of tree code from Duan, Johnston, Lindsay, and Krasny.
  • Packages from Jacob White's group
    Including precorrected FFT, FastCAP, FastLap, FastHenry, etc.
