  • DASHMM
    The Dynamic Adaptive System for Hierarchical Multipole Methods is a general library implementing hierarchical multipole methods using the asynchronous multi-tasking HPX-5 runtime system. It provides a unified execution on both shared and distributed memory architectures. The package provides built-in implementations of 3D Laplace, Yukawa, and low-frequency Helmholtz potenials.
  • RECFMM
    RECFMM is a program representation and implementation of a recursive scheme for parallelizing the adaptive FMM on shared memory computers using Cilk runtime. RECFMM provides implementations of 3D Laplace and Yukawa potentials.
    Download: recfmm.tar
    Reference: B. Zhang, J. Huang, N. P. Pitsianisn, and X. Sun. RECFMM: Recursive Parallelization of the Adaptive Fast Multipole Method for Coulomb and Screened Coulomb Interactions. Commun. Comput. Phys. 20 (2), 534-550, 2016
  • KIFMM
    This kernel-independent FMM was developed by Ling, Biros and Zorin. It replaces the analytic expansions and translations in the FMM with equivalent density representations. The packages applies to second-order constant coefficient non-oscillatory elliptic PDEs.
